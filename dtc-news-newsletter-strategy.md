## Part 1: Newsletter Format & Structure

### The "DTC Money Minute" Framework

Combining Hormozi's "Mosy Money Minute" concept with Sabri's value-first principles. Brand the newsletter itself—make opening it a reward.

**Subject Line Strategy:**
- Use consistent branding: "DTC Money Minute #[X]: [lowercase curiosity hook]"
- Example: "DTC Money Minute #12: the product research trick nobody talks about"
- Lowercase everything after the colon (signals personal, not corporate)
- No emojis, no ALL CAPS, no letter case formatting

**Preview Text:**
- 24% of readers check preview text before deciding to open
- Make it a mini-hook, not "View in browser" garbage
- Example: "I found this breakdown on YouTube and had to share it..."

---

### The 5-Section Flow Structure

This structure takes 15-20 minutes to assemble once you have systems in place.

```
┌─────────────────────────────────────────────────────┐
│  SECTION 1: INSTANT REWARD (Quote/Tweet)            │
│  - First thing they see after opening               │
│  - Rewards them for clicking immediately            │
│  - High-performer tweets/quotes from e-com world    │
├─────────────────────────────────────────────────────┤
│  SECTION 2: WHAT'S WORKING NOW (Tactical Nugget)    │
│  - ONE concept, under 200 words                     │
│  - Problem → Solution → How-to                      │
│  - Curated from YouTube videos, podcasts, articles  │
├─────────────────────────────────────────────────────┤
│  SECTION 3: THE BREAKDOWN (Story-Sell Bridge)       │
│  - "I found this video where [creator] broke down   │
│     exactly how they [achieved result]..."          │
│  - Link to the source (trains clicks)               │
│  - Sets up the product recommendation naturally     │
├─────────────────────────────────────────────────────┤
│  SECTION 4: TOOL OF THE WEEK (Soft Sell)            │
│  - "Speaking of [topic], this tool makes it easy"   │
│  - NOT a hard pitch—frame as helpful resource       │
│  - Contextual to the content above                  │
├─────────────────────────────────────────────────────┤
│  SECTION 5: PS STATEMENT                            │
│  - Most-read part after subject line                │
│  - Meme, additional resource, or secondary CTA      │
│  - Can include affiliate link or lead magnet        │
└─────────────────────────────────────────────────────┘
```

## Part 5: Technical Setup

### Email Formatting Rules (Deliverability)

Per Hormozi and Sabri:

- **Plain text only** (no HTML templates, no logos)
- **No images** (except maybe PS meme as link, not embedded)
- **1-2 links maximum** per email
- **Avoid money language** in body ($$, FREE, BUY NOW)
- **Use Google-native links** when possible (YouTube, Google Docs)
- **Enable reply** - ask for engagement in first email
- **Easy unsubscribe** - don't hide it

### First Email After Joining Newsletter

```
Subject: welcome to DTC Money Minute (+ a gift)

Hey [Name],

You're in.

Starting now, you'll get tactical e-com stuff you can actually use.
Under 60 seconds to read. No fluff.

I also have something for you that isn't public anywhere.

Reply "SEND IT" and I'll shoot it over.

Talk soon,
[Name]

PS - If this ever stops being valuable, the unsubscribe is at the bottom.
No hard feelings. But I don't think you'll need it.
```

**Why this works:**
- Reply triggers engagement (better deliverability)
- Sets expectations for format
- Creates reciprocity before you ask for anything
- Easy unsubscribe builds trust
