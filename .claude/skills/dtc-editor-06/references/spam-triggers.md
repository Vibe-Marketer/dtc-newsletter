# Email Spam Triggers Reference
<!-- Used by dtcnews-editor skill for deliverability optimization -->

## How to Use This File
Spam filters assign "points" to suspicious words and patterns. Too many points = spam folder. Replace or remove these triggers to ensure inbox delivery.

## FORMATTING TRIGGERS

These formatting choices hurt deliverability regardless of words used.

### Always Avoid

| Pattern | Problem | Fix |
|---------|---------|-----|
| ALL CAPS WORDS | Screams spam | Use normal capitalization |
| Multiple exclamation points!!! | Looks desperate | Max 1 per email |
| $$$$ or $$$ | Financial spam signal | Spell out or use once |
| Excessive bold | Looks promotional | Max 3 bold phrases per section |
| Colored text | Looks like marketing | Use sparingly, dark colors only |
| Large fonts | Looks promotional | Keep consistent sizing |
| Too many links | Looks like spam | Max 3-5 links per section |
| Image-heavy email | Can't read = spam | Balance text and images |
| No plain text version | Spam signal | Always include plain text |
| URL shorteners | Hide destination = spam | Use full URLs |

### Subject Line Rules

| Avoid | Why | Do Instead |
|-------|-----|------------|
| ALL CAPS | Spam signal | Normal caps |
| RE: or FW: (fake) | Deceptive | Don't fake replies |
| $ signs | Financial spam | Spell out or omit |
| % symbols | Promotional | Write "percent" or omit |
| ! at start | Looks urgent/spammy | Start with letter |
| Multiple ? or ! | Desperate | One max |
| [brackets] | Looks automated | Remove or use naturally |
| emojis at start | Can trigger filters | Mid-subject OK |

---

## SUBJECT LINE CHECKLIST

Before sending, verify subject line:

- [ ] Under 50 characters (ideal: 30-40)
- [ ] No high-risk trigger words
- [ ] No ALL CAPS
- [ ] No $ or % symbols
- [ ] Max 1 punctuation mark
- [ ] Starts with letter (not emoji or symbol)
- [ ] Doesn't sound like a reply (RE:) unless it is
- [ ] Preview text complements (doesn't repeat) subject

## TESTING RECOMMENDATIONS

Before sending any campaign:

1. **Use mail-tester.com** - Free spam score check
2. **Send test to Gmail, Outlook, Yahoo** - Check all major providers
3. **Check promotions tab** - Gmail especially
4. **Review spam folder** - For test accounts
5. **Run through Litmus or Email on Acid** - Professional testing

Acceptable spam score: 9/10 or higher on mail-tester.com
