# Prompt Templates for Common Ecom Tasks

Ready-to-adapt templates for the most common DTCNews tactic categories.

---

## Product Description Generation

### Basic Template
```
I sell [YOUR PRODUCT]. Write 5 unique product descriptions for my website. Each should be 2-3 sentences, highlight a different benefit, and end with a soft call-to-action. Make them sound confident but not salesy.
```

### Benefit-Focused Variation
```
Write a product description for [YOUR PRODUCT]. Structure it as: 1) One-line hook about the main problem it solves, 2) Three bullet points with specific benefits (not features), 3) One sentence about who it's perfect for. Keep it under 100 words total.
```

### Premium/Luxury Variation
```
Write a product description for [YOUR PRODUCT] that justifies a premium price. Focus on craftsmanship, materials, and the feeling of ownership. Avoid words like "cheap" or "affordable." Under 75 words.
```

## Ad Copy Creation

### Facebook Headlines
```
Write 10 Facebook ad headlines for [YOUR PRODUCT]. Each headline should create curiosity without clickbait. Use these patterns: question headlines, 'how I...' headlines, counterintuitive statements, and specific number claims. Keep each under 10 words.
```

### Primary Text (Body Copy)
```
Write Facebook ad primary text for [YOUR PRODUCT]. Structure: Hook question → Problem agitation → Solution intro → Social proof placeholder → CTA. Keep it under 125 words. Make the first line stop the scroll.
```

### Google Ads
```
Write 5 Google Search ad variations for [YOUR PRODUCT]. Each needs: Headline 1 (30 chars, include main keyword), Headline 2 (30 chars, benefit), Description (90 chars, CTA). Focus on buyer intent, not awareness.
```

## Product Research

### Competitor Analysis
```
I'm considering selling [YOUR PRODUCT]. Analyze what customers love and hate about this product based on common Amazon review patterns. Give me: 1) Top 3 features customers rave about, 2) Top 3 complaints to avoid, 3) One product improvement that would beat competitors.
```

### Niche Validation
```
I want to sell products in the [YOUR NICHE] market. List: 1) 5 underserved customer pain points in this niche, 2) 3 product categories with room for new brands, 3) What would make someone switch from their current solution. Be specific, not generic.
```

### Bundle Ideas
```
I sell [YOUR PRODUCT]. Suggest 5 product bundles I could create. For each bundle: name it, list what's included, explain why customers would want it together, and suggest a discount percentage that still protects margin.
```

## Email Subject Lines

### Win-Back Campaign
```
Write 7 email subject lines to win back customers who haven't purchased from [YOUR BRAND] in 90+ days. Mix these approaches: curiosity, FOMO, direct offer, personal check-in, and 'we miss you' angles. Keep each under 50 characters. No spam words.
```

### Cart Abandonment
```
Write 5 cart abandonment email subject lines for [YOUR BRAND]. Progress from gentle reminder to urgency across the sequence. Subject line 1 should be soft, subject line 5 should create FOMO. Under 40 characters each.
```

### Product Launch
```
Write 7 email subject lines announcing [YOUR PRODUCT]. Create different angles: mystery/teaser, direct announcement, problem-solution, social proof, scarcity, and exclusive access. Mark which are best for VIPs vs. general list.
```

### Newsletter/Content
```
Write 10 email subject lines for a weekly newsletter from [YOUR BRAND]. Make readers curious enough to open without being clickbait. Mix formats: questions, lists, surprising facts, and direct value statements. Under 50 characters.
```

## Customer Service

### FAQ Generation
```
I sell [YOUR PRODUCT]. Write the 10 most common questions customers would ask before buying, and answers for each. Focus on: shipping, returns, product specs, usage, and comparisons to alternatives. Keep answers under 50 words each.
```

### Response Templates
```
Write 5 customer service response templates for [YOUR BRAND] handling: 1) Late shipment complaint, 2) Product not as expected, 3) Requesting a discount, 4) Positive review thank you, 5) Refund request. Sound human, not corporate. Include [PERSONALIZATION] placeholders.
```

## Social Media

### Content Ideas
```
Give me 30 days of social media post ideas for [YOUR BRAND]. Mix: product showcases, behind-the-scenes, user-generated content prompts, educational tips, and engagement questions. One line per idea. No generic "motivational quotes."
```

### Instagram Captions
```
Write 5 Instagram captions for [YOUR PRODUCT]. Each caption should: hook in the first line, tell a mini-story or share a tip, end with a question or CTA. Include relevant hashtag suggestions. Mix lengths: 1 short, 3 medium, 1 long.
```

---

## Usage Notes

When adapting these templates:
1. Keep the structural elements (numbered lists, specific counts)
2. Adjust the bracketed variable to match your tactic
3. Add one constraint that matches your Deep Dive's specific angle
4. Remove any section that isn't essential
5. Verify final word count is under 150
