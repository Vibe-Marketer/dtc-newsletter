Create execution/generators/prompt_pack.py:

1. Import json, BaseGenerator, ProductSpec, GeneratedProduct

2. Define PROMPT_PACK_STRUCTURE:
   - categories: list of category names (3-5 categories)
   - prompts_per_category: 5-10 prompts each
   - Each prompt has: title, prompt_text, expected_output_description, example_output

3. Define PROMPT_PACK_PROMPT for Claude to generate prompts:
   - Takes problem, solution_name, target_audience
   - Outputs structured JSON with categories and prompts
   - Each prompt should be specific, actionable, copy-pasteable
   - Include placeholders like [YOUR PRODUCT] for customization

4. PromptPackGenerator(BaseGenerator):
   - __init__(self, claude_client=None)
   - get_product_type() -> "prompt_pack"
   - generate(spec: ProductSpec) -> GeneratedProduct:
     - Uses Claude to generate categorized prompts
     - Returns GeneratedProduct with files:
       - "prompts.json": Structured prompt data
       - "prompts.md": Human-readable markdown version
       - "README.md": How to use the pack
       - "QUICK_START.md": 5 best prompts to try first
   - validate(product: GeneratedProduct) -> bool:
     - Checks at least 3 categories
     - Checks at least 15 total prompts
     - Checks each prompt has title and prompt_text
     - Checks no prompt is under 20 chars
   - _format_prompts_markdown(prompts_data: dict) -> str:
     - Formats prompts with headers, descriptions, copy blocks
   - _generate_quick_start(prompts_data: dict) -> str:
     - Selects 5 most impactful prompts
