# Quick Start: 5 Best Prompts

Don't know where to begin? Start with these 5 high-impact prompts.

## 1. Quick Assessment
*From: Getting Started*

```
I'm dealing with Pricing strategy changes take forever to implement across channels. Analyze my current situation: [DESCRIBE YOUR SITUATION]. What are the top 3 priorities I should focus on first?
```

**What you'll get:** A prioritized list of 3 focus areas with reasoning

## 2. Root Cause Analysis
*From: Problem Solving*

```
I'm experiencing [DESCRIBE PROBLEM] related to Pricing strategy changes take forever to implement across channels. Help me identify the root cause using the 5 Whys technique.
```

**What you'll get:** A 5 Whys analysis leading to root cause

## 3. Strategic Framework
*From: Strategy & Planning*

```
Create a strategic framework for E-commerce entrepreneurs to address Pricing strategy changes take forever to implement across channels over the next [TIME PERIOD]. Include goals, metrics, and milestones.
```

**What you'll get:** A comprehensive strategic framework

## 4. Action Plan Creator
*From: Getting Started*

```
Create a 30-day action plan for [YOUR SPECIFIC GOAL] related to Pricing strategy changes take forever to implement across channels. Include weekly milestones and daily tasks.
```

**What you'll get:** A structured 30-day plan with clear milestones

## 5. Solution Brainstorm
*From: Problem Solving*

```
Generate 10 creative solutions for [SPECIFIC CHALLENGE] that E-commerce entrepreneurs faces. Include both conventional and unconventional approaches.
```

**What you'll get:** 10 solution ideas ranging from simple to creative
