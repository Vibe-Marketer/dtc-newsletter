## Usage

Run 'python cart_abandonment_analyzer.py <website_url>' to analyze a site for cart abandonment factors. Use --export to save results to JSON, --timeout to set request timeout. The script analyzes load times, mobile friendliness, checkout complexity, payment options, and provides actionable recommendations to reduce cart abandonment rates.

For full options, run:
```bash
python idea_cart_abandonment_automator.py --help
```
