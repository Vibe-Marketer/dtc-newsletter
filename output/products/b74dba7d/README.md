## Usage

Save as 'shipping_compare.py' and run:

1. SINGLE PACKAGE: python shipping_compare.py --weight 2.5 --dimensions 10x8x6 --origin 90210 --dest 10001
2. BATCH MODE: python shipping_compare.py --csv packages.csv --output results.csv
3. EXPRESS SERVICE: python shipping_compare.py --weight 5 --dimensions 12x10x8 --origin 60601 --dest 30301 --service express

For CSV batch processing, create a CSV with columns: weight, dimensions (LxWxH), origin_zip, dest_zip, service_type (optional).

The script calculates estimated rates for USPS, UPS, and FedEx using simplified algorithms. For production use, replace with actual carrier APIs by adding your API keys via --api-keys config.json file.

For full options, run:
```bash
python shipping_rate_comparison_automator.py --help
```
