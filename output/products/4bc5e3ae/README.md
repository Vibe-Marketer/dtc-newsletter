   ```bash

   ```

## Usage

1. Prepare your sales data in CSV format with columns: date (YYYY-MM-DD), product_id, quantity_sold
2. Run the script: python inventory_forecaster.py sales_data.csv
3. Optional parameters:
   - -o forecast_report.csv: Specify output file
   - --forecast-days 45: Set forecast period
   - --safety-factor 2.0: Adjust safety stock multiplier
   - --lead-time 14: Set supplier lead time in days
4. The script outputs a CSV with optimal stock levels, reorder points, and safety stock recommendations for each product
5. Use the generated recommendations to set inventory levels and reorder triggers in your e-commerce system

For full options, run:
```bash
python inventory_forecasting_pure_automator.py --help
```
